Summer Program 2026

Speed & Agility Class

A 2-day summer intensive class for children to improve running, stopping, turning, and reaction skills in a fun and safe environment.

Ages 6–12

Build Better Movement for Sport

Children do not improve speed only by running more. Good posture, balance, rhythm, reaction, and body control all matter. This program helps children experience these basic movement skills in an enjoyable and practical way.

Recommended For Children Who…

Are new to sports

Children can learn basic movement skills in a supportive and enjoyable setting.

Want to move with more confidence

We help children improve body control, first-step movement, stopping, and changing direction.

Main Program Content

Day 1

Run, React, and Move with Rhythm

Warm-up, coordination drills, running basics, reaction starts, and speed games.

Day 2

Stop, Turn, and Balance

Foot balance, stopping mechanics, turning drills, zig-zag movement, and agility challenges.

Coaching Focus

Fun, Safe, and Positive Learning

We value each child’s improvement and confidence rather than comparing children against each other.
